# Project Hub 

## one stop destination for the tracking projects and user status on tasks 

Features :

1) Admin page for all viewing and creating projects and assigning tasks to emplyees
2) Employee page for viewing the employee based tasks and allows for updation of the tasks.
3) Models for admins , employees , projects and tasks to reuse for forms and templates .
4) CRUD operations for the deletion of tasks and projects and users assigned to the tasks ensuring the intigrity constraints are maintained .

Simple and efficient view model for the easy access. 

## Steps to run on local :
1) copy the project via : "git clone https://github.com/kkumar360/projecthub.git"
2) install packages via the pip command : pip install django, django-admin-import-export,request
3) run : cd project_planner
4) run : python manage.py migrate
5) run : python manage.py createsuperuser -----> create a superuser admin to access the django admin panel
6) run : python manage.py runserver host:port -----> this runs the server
